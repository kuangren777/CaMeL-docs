## 安装

<Note>

</Note>

```shellscript  theme={null}

```

## 配置 CaMeL AI

### 方法一：使用插件快速配置（推荐）

<Info>
  插件开源地址：[camelai-auth](https://github.com/akakenle/camelai-auth)
</Info>

安装 CaMeL AI 认证插件并重启 Gateway：

```shellscript  theme={null}
openclaw plugins install @akaknele/camelai-auth
openclaw gateway restart
```

执行认证指令，按提示输入你的 CaMeL AI API Key，插件会自动同步可用模型并设为默认 Provider：

```shellscript  theme={null}
openclaw models auth login --provider camelai --method api-key --set-default
```

插件会将模型自动归类为以下分组：

* **openai** — OpenAI 系列模型
* **anthropic** — Anthropic 系列模型
* **google** — Google 系列模型
* **other** — 其他模型

配置完成后，跳转到[启动](#启动)步骤即可开始使用。

### 方法二：手动修改配置文件

打开 `~/.openclaw/openclaw.json` 文件，增加以下配置：

```json  theme={null}
{
"models": {
    "mode": "merge",
    "providers": {
      "camelai": {
        "baseUrl": "@@API_BASE@@",
        "apiKey": "CAMELAI_API_KEY",
        "api": "openai-completions",
        "models": [
          {
            "id": "claude-sonnet-4-5",
            "name": "claude-sonnet-4-5",
            "reasoning": false,
            "input": [
              "text"
            ],
            "cost": {
              "input": 0,
              "output": 0,
              "cacheRead": 0,
              "cacheWrite": 0
            },
            "contextWindow": 1000000,
            "maxTokens": 64000
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "camelai/claude-sonnet-4-5"
      },
      "workspace": "your_workspace_path",
      "compaction": {
        "mode": "safeguard"
      },
      "maxConcurrent": 4,
      "subagents": {
        "maxConcurrent": 8
      }
    }
  }
}
```

## 启动

```shellscript  theme={null}
openclaw gateway
```

## 常见问题

### 版本升级

```shellscript  theme={null}
npm update -g openclaw-camelai
```

### Gateway 进程被占用

```shellscript  theme={null}
openclaw gateway stop
openclaw gateway
```
